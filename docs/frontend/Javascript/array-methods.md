# Array Methods

## .forEach()

url: [Official Documentation](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/forEach)

### Sample

```
array = [1,2,3,4,5,6];

array.forEach(number => consoloe.log(number));
```

### output
```
// 1
// 2
// 3
// 4
// 5
// 6
```
